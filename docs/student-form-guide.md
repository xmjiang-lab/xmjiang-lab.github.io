# 蒋晓鸣课题组 · 表单使用指南（投稿者）

本指南面向想要加入课题组的学生、即将离任的课题组成员，以及希望投稿
课题组新闻的合作者。

课题组主页：<https://xmjiang-lab.github.io/>

---

## 我应该填哪个表单？

| 我的身份 / 目的 | 表单 | 入口 |
|---|---|---|
| 我想申请加入课题组（本科 / 硕士 / 博士） | 学生申请 | <https://xmjiang-lab.github.io/student-form.html> |
| 我是课题组毕业生 / 离任成员，想登记最新去向 | 毕业 / 离任 | <https://xmjiang-lab.github.io/graduation-form.html> |
| 我想投稿一条课题组新闻 / 招募 / 公告 | 新闻投稿 | <https://xmjiang-lab.github.io/news-form.html> |

每个表单页面右上角都有「← 返回 …」链接，迷路时可回到主站。

---

## 一、学生申请 (student-form.html)

### 必填字段

- **中文姓名**
- **申请层次**（博士 / 硕士 / 本科）
- **入组年份**：例如 2026
- **中文简介**：2–3 句话即可，可在审核通过后再完善
- **教育背景**：用 `|` 分隔多段，每段一行；例如：
  > BA, X University (2019–2023) | MA, Y University (2023–2026)
- **研究方向**：用逗号分隔，例如 `vocal emotion, ERP, social cognition`

### 选填字段

- 英文姓名 / 英文简介
- 邮箱、个人主页、ORCID、Google Scholar
- 头像（建议 400×400 jpg/png，**不超过 2 MB**）

提交成功后会看到一条「Submitted」提示和 GitHub Issue 链接。蒋老师审核通过
后，您的资料会自动出现在「团队成员」页面。审核通常在几个工作日内完成。

---

## 二、毕业 / 离任登记 (graduation-form.html)

### 关键点

- **中文姓名必须与「团队成员」页面上的现任成员姓名完全一致**，否则系统无法匹配。
  如果发现不一致，请联系 Wenjun 协助修复。
- **离组日期**：填入您正式离开课题组的日期，例如 `2026-06-30`。
- **在组时间段**：例如 `2023–2026`。
- **现任职位（中文）**：例如 `麦吉尔大学 博士研究生`。
- **现任职位（英文）**（可选）：例如 `PhD candidate, McGill University`。

提交后将在 GitHub 创建一条 Issue。蒋老师审核通过后，您会自动从「现任成员」
栏移至「毕业生 / 离任成员」栏。

---

## 三、新闻投稿 (news-form.html)

### 必填字段

- **中文标题**
- **中文标签**：常见标签包括「国际合作招募」「博士招生」「学术活动」等
- **日期**：YYYY-MM-DD 格式
- **中文正文**：1–2 段，介绍新闻内容

### 选填字段

- 英文标题 / 英文标签 / 英文正文
- 外部链接（如指向 SISU、STCSM 等官方公告页面）

新闻按日期倒序排列。审核通过后会显示在「新闻动态」页面，最近 3 条还会
出现在首页右侧的「近期动态」栏。

---

## 四、提交失败怎么办

表单依赖 GitHub API。如果出现「提交失败 / Submission failed」错误，常见原因：

1. **网络问题**：刷新页面后重试。
2. **网站尚未配置 Token**：刚部署的新站点可能还没有配置 GitHub Token，
   此时表单不可用。请直接邮件投稿。
3. **照片过大**：学生申请的头像不要超过 2 MB。压缩后再试。
4. **GitHub API 限流**：如同一 IP 短时间内大量提交，可能被临时限流，
   稍后再试。

任何情况下，**都可以改用邮件**，直接发送内容到：

> <xiaoming.jiang@shisu.edu.cn>

## 五、隐私 / 数据处理说明

- 表单提交会在 GitHub 创建一条 **公开** 的 Issue。
- 审核通过后，您填写的姓名、简介、教育背景、研究方向、邮箱等会出现在
  课题组主页（公开可访问）。
- 头像同样会显示在主页。
- 不希望公开的字段，请不要在表单中填写。
- 您可以在审核前或审核后随时联系蒋老师或 Wenjun 申请删除自己的信息。
